# UM Interior Backend

## Live API

https://u-m-interior-backend.onrender.com

## Project Overview

UM Interior Backend is a Node.js and Express.js based REST API that powers the UM Interior Management System. The application handles user management, authentication, project management, and data storage using MongoDB.

## Features

* User Authentication and Authorization
* RESTful API Architecture
* MongoDB Database Integration
* CRUD Operations
* Middleware-based Request Handling
* Error Handling and Validation
* Frontend Integration Support
* Secure Environment Variable Configuration

## Tech Stack

### Backend

* Node.js
* Express.js

### Database

* MongoDB
* Mongoose

### Other Tools

* JWT Authentication
* Render Deployment
* Git & GitHub

## Project Structure

config/ - Database and application configuration

controllers/ - Business logic and request handling

middleware/ - Authentication and custom middleware

models/ - Database schemas and models

routes/ - API routes

views/ - Server-rendered views (if applicable)

public/ - Static assets

app.js - Application entry point

## Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Create a .env file and add the required environment variables.

Start the application

```bash
npm start
```
